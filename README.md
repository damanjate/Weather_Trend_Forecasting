
# Weather Trend Forecasting

## Project Overview

This project analyzes and forecasts global weather trends using machine learning and environmental analytics on the Global Weather Repository dataset.

The project includes data cleaning, exploratory data analysis (EDA), anomaly detection, climate analysis, air quality analysis, feature importance analysis, spatial analysis, and forecasting model comparison.

## PM Accelerator Mission

Empowering professionals through AI, machine learning, and real-world project development.

## Dataset

Source: Global Weather Repository Dataset (Kaggle)

* 141,898 weather observations
* Global weather data from cities around the world
* More than 40 weather and environmental features

## Project Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Correlation Analysis
* Time Series Analysis using `last_updated`
* Anomaly Detection
* Climate Analysis
* Environmental Impact Analysis
* Spatial & Geographical Analysis
* SHAP Explainability
* Forecasting Model Comparison
* Ensemble Learning

## Models Evaluated

1. Linear Regression
2. Random Forest
3. Tuned Random Forest
4. Bagging Regressor
5. Gradient Boosting
6. AdaBoost
7. XGBoost
8. Ensemble Model (Random Forest + XGBoost)

## Best Model

**Ensemble Model (Random Forest + XGBoost)**

Performance:

* RMSE: 4.71
* MAE: 3.19
* R² Score: 0.765

## Evaluation Metrics

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score
