# Weather_Trend_Forecasting
Weather Trend Forecasting

## Objective
Analyze and forecast global weather trends using machine learning and time series techniques.

## Dataset
Global Weather Repository dataset from Kaggle.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Forecasting Models
- Ensemble Learning
- Climate Analysis
- Air Quality Analysis
- Spatial Analysis
- Feature Importance

## Models Used
- Linear Regression
- Random Forest
- Gradient Boosting
- XGBoost
- Ensemble Model

## Evaluation Metrics
- MAE
- RMSE
- R² Score
